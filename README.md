### 这是什么？

一款阿里云盘的目录文件列表程序，后端基于`golang`最好的`http`框架`gin`，前端使用`vue`和`ant design`。

### 前端项目地址

- https://github.com/Xhofe/alist-web

### 支持的功能

- 自动刷新`token`，`refresh_token`自动更新，失效时间未知，本人使用过程中没有失效过。
- 图片、视频、音频预览
- 目录加密
- `Readme`渲染
- 自定义根目录
- 文件直链下载
- …
